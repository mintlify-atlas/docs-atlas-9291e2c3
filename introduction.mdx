    Quick start
  </h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">
    Get up and running in minutes with this production-ready template
  </p>

  <Steps>
    <Step title="Clone the template">
      Use degit to clone the template without git history:
      
      ```bash
      npx degit w3cj/hono-open-api-starter my-api
      cd my-api
      ```
    </Step>

    <Step title="Install dependencies and configure">
      Install dependencies and set up your environment:
      
      <CodeGroup>
      ```bash npm
      npm install
      cp .env.example .env
      ```
      
      ```bash pnpm
      pnpm install
      cp .env.example .env
      ```
      
      ```bash yarn
      yarn install
      cp .env.example .env
      ```
      </CodeGroup>
    </Step>

    <Step title="Initialize the database">
      Create your SQLite database and push the schema:
      
      ```bash
      pnpm drizzle-kit push
      ```
    </Step>

    <Step title="Start the development server">
      Run the development server with hot reload:
      
      ```bash
      pnpm dev
      ```
      
      Your API is now running at `http://localhost:3000`
      
      <Accordion title="View available endpoints">
        - **GET /doc** - OpenAPI specification (JSON)
        - **GET /reference** - Interactive Scalar API documentation
        - **GET /tasks** - List all tasks
        - **POST /tasks** - Create a new task
        - **GET /tasks/:id** - Get a task by ID
        - **PATCH /tasks/:id** - Update a task
        - **DELETE /tasks/:id** - Delete a task
      </Accordion>
    </Step>
  </Steps>
</div>
