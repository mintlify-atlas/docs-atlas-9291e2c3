## Endpoint

<ParamField path="GET" type="/tasks">
  Retrieves all tasks from the database
</ParamField>

## Request

This endpoint does not require any parameters or request body.

### Example Request

<CodeGroup>

```bash cURL
curl -X GET http://localhost:9999/tasks
```

```javascript JavaScript
const response = await fetch('http://localhost:9999/tasks', {
  method: 'GET',
  headers: {
    'Content-Type': 'application/json'
  }
});

const tasks = await response.json();
console.log(tasks);
```

```python Python
import requests

response = requests.get('http://localhost:9999/tasks')
tasks = response.json()
print(tasks)
```

</CodeGroup>

## Response

<ResponseField name="tasks" type="array">
  An array of task objects

  <Expandable title="Task Object">
    <ResponseField name="id" type="number" required>
      The unique identifier for the task
    </ResponseField>

    <ResponseField name="name" type="string" required>
      The name of the task
    </ResponseField>

    <ResponseField name="done" type="boolean" required>
      Whether the task is completed
    </ResponseField>

    <ResponseField name="createdAt" type="timestamp">
      The date and time when the task was created
    </ResponseField>

    <ResponseField name="updatedAt" type="timestamp">
      The date and time when the task was last updated
    </ResponseField>
  </Expandable>
</ResponseField>

### Status Codes

<ResponseField name="200" type="OK">
  Successfully retrieved the list of tasks
</ResponseField>
